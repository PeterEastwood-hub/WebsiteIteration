Victory Native

A charting library for React Native with a focus on performance and customization.

  • Documentation
  • Github

Features

  • FlexibleFully contained, reusable data visualization elements are responsible for their own styles and behaviors.
  • Easy to UseComprehensive component and hook based elements make setting up and configuring a custom chart simple.
  • PerformantA foundation for building high-end data visualizations that can animate at over 100 FPS even on low-end devices.
